== Description ==

Eliza, named after that famous psychoterapist, let's you create a magazine website with a clean and modern design. Eliza comes with two custom widgets, In This Section, which lists subcategories in the category archive pages; and Posts from Category, which shows the latest posts from a specific category. Make it yours with a custom logo, two widget areas and a custom menu.

For more information about Eliza please go to <https://arthr.me/themes/eliza/>.

== Installation ==

1. Download the latest version from the theme from this repository (soon on WordPress Theme Directory!).
2. Uploads the theme folder in your `wp-content/themes`.
3. Click on “Activate” button to use your theme.
4. Or click in the “Preview” button to open the Customize panel and make it yours before activating it.
